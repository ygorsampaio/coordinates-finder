# 📍 Localizador de Coordenadas (GeoFinder)

Este projeto foi desenvolvido como um desafio técnico para criar uma aplicação funcional que consome dados de uma API real. O objetivo é permitir que o usuário encontre a latitude e longitude de qualquer localidade de forma rápida e simples.

## 📱 Sobre o Projeto
O sistema permite buscar endereços, cidades ou pontos turísticos. Ele utiliza a API do OpenStreetMap para retornar as coordenadas geográficas e oferece um link direto para visualizar o local no Google Maps.

**Diferenciais:**
- Layout focado em dispositivos móveis (Mobile-First).
- Interface limpa e intuitiva.
- Consumo de API pública sem necessidade de chaves privadas (Open Source).

## 🛠️ Tecnologias Utilizadas
- **HTML5**: Estrutura semântica.
- **CSS3**: Estilização moderna e responsividade.
- **JavaScript (JS)**: Lógica de consumo da API e manipulação do DOM.
- **API Nominatim (OpenStreetMap)**: Serviço de geocodificação gratuito.

## 🚀 Como testar
1. Digite um local no campo de busca (Ex: "Cristo Redentor" ou "Avenida Paulista").
2. Clique em **Pesquisar**.
3. O resultado aparecerá logo abaixo com o endereço completo e as coordenadas.
4. Clique em **Ver no Mapa** para abrir a localização exata no Google Maps.
